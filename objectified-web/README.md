# Objectified Marketing Website

The official marketing website for Objectified - a visual OpenAPI 3.1.0 specification editor and database schema designer.

## Features

- Home page with product overview
- Features showcase
- Pricing information
- Contact form
- Sign up page
- Sign in page
- Community resources
- Documentation links
- Responsive design with dark mode support

## Getting Started

Install dependencies:

```bash
yarn install
```

Run the development server:

```bash
yarn dev
```

Open [http://localhost:3002](http://localhost:3002) in your browser.

## Build

```bash
yarn build
yarn start
```

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Radix UI
- Lucide Icons

## Structure

- `/src/app` - Next.js app directory with pages
- `/src/app/components` - React components
- `/src/lib` - Utility functions and helpers
- `/public` - Static assets

## Development

This is part of the Objectified monorepo. The marketing website runs on port 3002 to avoid conflicts with other services.
