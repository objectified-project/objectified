Design notes & colour system

  • Indigo — primary brand colour, current-tenant highlight, KPI cards
  • Violet — Admin role badge, member management panel
  • Emerald — Enabled status, verified user indicator
  • Rose — Disabled tenant, danger zone, remove actions
  • Current tenant is always shown first via a hero card above the table.
  • Slug changes surface an inline amber warning in the edit form before submission.
  • Admin-only controls (Edit, Manage, Danger zone) are hidden / greyed for non-admins.
  • Tenant detail is a full page (replaces the fragile hidden inline panel in the current impl).
