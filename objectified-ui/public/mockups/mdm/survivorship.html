Custom expression · website

Sandboxed JEXL · access to candidates[] and refs.*

Expression

// Pick the canonical URL form from any source // Strategy: prefer https + www, fall back to source_rank let values = candidates .filter(c => c.website != null) .map(c => ({ src: c.source, rank: refs.source_rank[c.source], url: canonical_url(c.website) })); if (values.length == 0) return null; let https = values.filter(v => v.url.startsWith("https://")); let pool = https.length > 0 ? https : values; return pool.sort((a, b) => a.rank - b.rank)[0].url;
