POST /api/v1/orders tag: Orders

Create a new order. Returns the created entity with a generated id, created_at, and an idempotency-safe trace_id.

Request body · application/json

UUID of the customer placing the order. Must reference an existing record.

example "3f2a4c1b-…"

Responses

201Ordercreated
422ProblemDetailsvalidation failed
429RateLimitProblemquota exceeded
GET /api/v1/orders/{id} Retrieve a single order by id
PATCH /api/v1/orders/{id} Partial update of an order
DELETE /api/v1/orders/{id} Cancel and remove order 
import { OrdersApi } from '@acme/sdk';

const orders = new OrdersApi({
  apiKey: process.env.ACME_KEY,
});

const order = await orders.create({
  customer_id: '3f2a4c1b-…',
  items: [{ sku: 'WIDGET-1', qty: 2 }],
  currency: 'USD',
});

console.log(order.id);
