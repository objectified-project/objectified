Installation

Install the package with your preferred package manager. The SDK ships with full TypeScript typings.

npm pnpm yarn 
$ npm install @acme/orders-sdk

Authentication

All requests require a Bearer token. Tokens scoped to orders:read and orders:write are issued from the portal.

import { OrdersClient } from "@acme/orders-sdk";

const client = new OrdersClient({
  apiKey: process.env.ACME_API_KEY,
  baseUrl: "https://api.acme.com/v3",
  retries: 3,
});

Quickstart

Create an order, then retrieve it. Errors throw typed exceptions you can pattern-match against.

// Create an order
const order = await client.orders.create({
  customerId: "cus_8421",
  items: [{ sku: "WIDGET-A", qty: 2 }],
});

// Retrieve it
const fetched = await client.orders.retrieve(order.id);

try {
  await client.orders.cancel("ord_unknown");
} catch (e) {
  if (e instanceof NotFoundError) handle(e);
}

API reference · OrdersClient

.list(params?: ListOrdersParams) → Promise<PaginatedOrders>

List orders with pagination and filters (status, date range, customer).

.create(body: NewOrder) → Promise<Order>

Create a new order. Throws ValidationError on a malformed body.

.retrieve(orderId: string) → Promise<Order>

Fetch an order by its id. Throws NotFoundError if missing.

.cancel(orderId: string) → Promise<Order>

Cancel an order. Idempotent — cancelling a cancelled order returns the same payload.
