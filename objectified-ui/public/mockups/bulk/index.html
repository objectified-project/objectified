Objectified Bulk

BULK INGESTION

Big data bulk loader for object stores and data lakes. Ingests Parquet, Avro, ORC, Delta Lake, CSV, and JSON Lines from S3, GCS, Azure Blob, HDFS, and local uploads directly into objectified-db. Chunked upload with resume, partition management, and multi-file fanout. DataFusion for single-node, Spark for cluster-scale, Arrow Flight as the high-throughput transport layer.

DB Hooks: load_jobs batch_file_sources data_pipeline_runs classes tenants
