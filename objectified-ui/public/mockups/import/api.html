Headless Import API & CI/CD

REST endpoints · objectified-cli · GitHub Actions · GitLab CI · webhook triggers

REST endpoints · v1

Auth: Authorization: Bearer <api-key> · scope import:create

  • POST/api/v1/importscreate & start a new import job
  • GET /api/v1/imports/{jobId}poll status & progress
  • GET /api/v1/imports/{jobId}/streamSSE / WebSocket progress stream
  • POST/api/v1/imports/{jobId}/commitcommit a previewed import
  • POST/api/v1/imports/{jobId}/rollbackrollback a committed import
  • DEL /api/v1/imports/{jobId}cancel an in-flight import
  • POST/api/v1/imports/ddl-analyzeparse SQL DDL · returns schemas preview
  • POST/api/v1/imports/webhook/{tokenId}webhook-triggered import (CI / Git push)

objectified-cli

Local + CI · npm / brew / scoop

v1.4.0 
# install
$ npm install -g @objectified/cli

# auth (writes ~/.objectified/config)
$ objectified login --api-key=$OBJ_KEY

# import
$ objectified import \
  --file openapi.yaml \
  --project acme-platform \
  --version v3.3-rc1 \
  --preset acme-default \
  --wait --json

# dry-run + diff vs latest
$ objectified import --file openapi.yaml --dry-run --diff

GitHub Actions

Auto-import on push to main

objectified/import-action@v1 
# .github/workflows/import.yml
name: Import OpenAPI to Objectified
on:
  push:
    branches: [main]
    paths: ["specs/**.yaml"]

jobs:
  import:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: objectified/import-action@v1
        with:
          api-key:    ${{ secrets.OBJ_API_KEY }}
          project:    acme-platform
          version:    ${{ github.sha }}
          file:       specs/openapi.yaml
          preset:     acme-default
          wait-for-completion: true
          fail-on-warnings:    false
          comment-on-pr:       true

GitLab CI

Pipeline-triggered import

includes/objectified.yml 
# .gitlab-ci.yml
include:
  - remote: "https://templates.objectified.dev/gitlab/import.yml"

import_to_objectified:
  extends: .objectified-import
  rules:
    - if: $CI_COMMIT_BRANCH == "main"
  variables:
    OBJ_PROJECT:  acme-platform
    OBJ_VERSION:  $CI_COMMIT_SHA
    OBJ_FILE:     specs/openapi.yaml
    OBJ_PRESET:   acme-default
    OBJ_DRY_RUN:  "false"
  artifacts:
    when: always
    paths: [import-report.html]
