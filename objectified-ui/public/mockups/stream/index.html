Objectified Stream

STREAM INGEST

Real-time streaming ingest from Apache Kafka, AWS Kinesis, Apache Pulsar, and MQTT into objectified-db. Maps topics to Classes with configurable field mapping rules and schema evolution policies. Supports schema-on-read and schema-on-write modes, micro-batching with configurable flush intervals, per-partition lag monitoring, and a dead-letter queue for failed events.

DB Hooks: stream_topics stream_consumer_groups load_jobs classes tenants
