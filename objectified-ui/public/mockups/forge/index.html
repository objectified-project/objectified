Forge mockups

Visual designs for the drag-and-drop DAG pipeline studio — source connectors, transformation steps, target loaders, A/B testing of transform variants, and an inline row-level debugger. Pipelines are versioned alongside objectified schemas; changing a class definition flags dependent pipelines for review. Execution engine is DataFusion for embedded workloads and Apache Spark for cluster-scale payloads. Mockups follow the Objectified Studio design system: Inter typography, orange accent, slate gray scale, and dark mode parity.
