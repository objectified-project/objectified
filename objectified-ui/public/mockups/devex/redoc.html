GET /users/{id} deprecated field

Get a user

Retrieves a single user by ID. The response includes the user's profile, contact, and address attributes. Authentication via bearer token is required.

Path parameters

id required string · uuid

Unique identifier for the user.

Response 200

Returns a User object on success.
