REST API

Trigger an export programmatically

curl -X POST \
  https://api.objectified.dev/v1/insights/exports \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "report":   "portfolio.health",
    "format":   "parquet",
    "filters":  { "team": "payments" },
    "delivery": {
      "type":   "s3",
      "bucket": "obj-reports",
      "prefix": "weekly/health/"
    },
    "callback": "https://hooks.acme.com/exports"
  }'
Returns a job id and a signed status URL. Webhook fires on completion with HMAC signature.
