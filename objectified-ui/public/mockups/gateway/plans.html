Sliding-window algorithm

Redis sorted sets store request timestamps per API key. The window slides continuously, eliminating the boundary spikes of fixed windows.

trailing 1s window · drops oldest as time advances

  • Accurate at boundaries (no fixed-window spikes)
  • Survives gateway restarts (Redis-backed)
  • Returns 429 + X-RateLimit-Remaining
