Design notes & colour system

  • Indigo / Purple — primary brand, project list, card accents
  • Violet — create flow, AI Design tab accent
  • Emerald — quality score A tier, enabled status, Published
  • Amber — In review, unsaved changes banner, quality score B tier
  • Rose — Deleted, danger zone, low quality C tier
  • Quality score tiers: A (≥80, emerald), B (60–79, indigo), C (<60, amber)
  • Domain category pills (violet) shown in project name cell — classifies schema domain
  • Slug auto-generates from project name but is independently editable
  • Create project uses a Starting Template picker that pre-fills OpenAPI metadata fields
