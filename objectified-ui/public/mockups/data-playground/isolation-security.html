Defense in depth

Network layer
enforced

Sandbox API endpoints cannot make outbound calls to production services. Egress allowlist scoped to api.objectified.io & objectstore.local.

Auth & session
enforced

Every request validates the sandbox session token. Expired or invalid tokens return 401 Unauthorized. Token TTL matches sandbox TTL.

Database layer
enforced

Each sandbox uses a dedicated PostgreSQL role with grants only on sandbox_{id}.*. RLS prevents cross-sandbox row access. Verified by 8 automated tests.
