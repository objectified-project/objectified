Profile — Design Mockups

Enterprise-grade improvements to the user profile & settings area

The current profile page exposes only basic account info and a password-change dialog. These mockups expand it into a full-featured personal settings hub: rich identity management, a granular security centre, a notification preference matrix, and a sessions audit view — all following the established Objectified design system.

Design notes

Colour system

  • Indigo — primary brand, identity & preferences
  • Emerald — security, healthy/hardened states
  • Amber — notifications, warnings, expiry
  • Sky — sessions, devices, connectivity
  • Rose — destructive actions, anomalies

Improvements over current implementation

  • Avatar upload with crop support
  • Passkey / WebAuthn registration (currently no MFA UI)
  • Dedicated security status panel with policy compliance
  • Full notification matrix (currently no notification prefs)
  • Quiet hours & digest cadence controls
  • Session audit log with anomaly detection callouts
  • Breach monitoring via HaveIBeenPwned integration
  • Per-tenant notification overrides
