Objectified File Gateway

FILE INGESTION

Object store and file system bridge for structured file ingestion into objectified-db. Connects to S3, GCS, Azure Blob, SFTP, and NFS. Monitors for new file arrivals via event notifications or polling, infers schema from file headers, deduplicates by content hash, and routes files to the appropriate Forge pipeline or direct loader via a configurable routing rules engine.

DB Hooks: batch_file_sources load_jobs data_source_connections classes api_keys

Storage & Connections

Storage Connections

S3 · GCS · Azure Blob · SFTP · NFS

Manage credentials and connection settings for cloud object stores, SFTP servers, and NFS mounts. Configure event notification vs polling watch mode per bucket.
Bucket Browser

Explore buckets, folders & file listings

Browse connected storage sources, preview file listings with size/modified timestamps, inspect individual file metadata, and manually trigger ingestion for specific files.
Schema Inference

Header sampling & type detection

Sample file headers to infer column names, data types, nullable flags, and schema fingerprints. Compare inferred schema against registered objectified Classes to suggest routing targets.

Format & Parsers

Format Configurator

CSV · Parquet · Avro · XML · Fixed-width

Configure format-specific parsing options: CSV delimiters and quoting, Parquet row-group size, Avro codec, XML namespace mappings, fixed-width column definitions.
Plugin Parsers

Excel · EDI · Fixed-width mainframe

Install and configure format plugins for Excel workbooks, EDI 850/856/810 documents, and legacy fixed-width mainframe extracts. Preview parsed output before enabling.

Routing & Deduplication

Routing Rules ★ Star

Pattern → Class routing engine

3-pane rule builder: priority-ordered rule list, pattern editor (filename glob, path, schema fingerprint → target Class + action), and live test panel.
Deduplication

Content hash & duplicate detection

Content-hash fingerprint log for every ingested file. Prevents double-loading files arriving from multiple sources or on retry. View suppressed duplicates and override when needed.

Monitoring & History

Arrival Monitor

Real-time file arrival feed

Live feed of new file arrivals from event notifications (S3 EventBridge, GCS Pub/Sub) and polling. Shows file name, size, source, routing outcome, and time-to-ingest.
Load History

File ingestion audit trail

Complete audit trail of every file ingested: source, format, rows loaded, target Class, routing rule matched, dedup status, and final load job outcome.
