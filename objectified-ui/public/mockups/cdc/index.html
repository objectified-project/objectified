Objectified CDC

CHANGE DATA CAPTURE

Captures row-level change events (inserts, updates, deletes) from PostgreSQL WAL, MySQL binlog, MongoDB oplog, Oracle LogMiner, and SQL Server CDC — streaming them into objectified-db with exactly-once delivery. Powered by Debezium, Maxwell's Daemon, or PgLogical. Includes schema DDL change handling, initial snapshot backfill, and a real-time lag monitor.

DB Hooks: cdc_slots cdc_schema_events load_jobs data_source_connections classes
