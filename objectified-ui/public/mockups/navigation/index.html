A header that scales to the whole suite

Today the global TopHeader carries four nav items in a center rail: Home, Control Panel, Designer, Paths. The suite already ships 23+ application surfaces (Architect, Connect, MDM, Data Shield, Data Transform, Pact, Contracts, Code Gen, Detective, Linting, Automation, Collaboration, Browser, DevEx, Data Insights, Analytics, Academy …) and the rail can't grow with it. Switching between apps requires detouring back to /ade and clicking a launcher tile.

This redesign collapses the rail into a single app pill on the left that opens a grouped, searchable app switcher. The center of the header becomes project / branch / tenant context — the things you actually need to see while you work. Sub-navigation for the current app moves to a thin second tier directly under the header. Visual system reuses every existing token: Inter typography, slate gray scale, indigo accent, panel cards, Lucide icons, persisted dark mode.
