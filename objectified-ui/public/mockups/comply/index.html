Comply mockups

Visual designs for Objectified Comply — regulatory compliance mapping and certification built directly into the schema design layer. Map class objects and API paths to GDPR, HIPAA, PCI-DSS, SOC 2, and ISO 27001 controls. Tag fields as PII, PHI, or PCI. Auto-generate auditor-ready evidence packages. Track which schema versions were in scope during an audit window. Bridges engineering and legal/risk teams in a single governed workspace. Mockups follow the Objectified Studio design system — Inter typography, teal accent, slate gray scale, panel-based layout, and dark mode parity.
