Authorization policy

valid
package objectified.gateway

# Same policy applies regardless of protocol — claims are normalized
# into `input.subject` at the listener.

default allow = false

allow {
  input.subject.tenant == input.resource.tenant
  some role in input.subject.roles
  role in {"admin", "writer"}
}

allow {
  input.method.protocol == "graphql"
  input.method.operation == "query"
  "reader" in input.subject.roles
}

deny[msg] {
  input.method.protocol == "asyncapi"
  not input.subject.scopes[_] == "events:publish"
  msg := sprintf("publish on %v requires events:publish", [input.resource.channel])
}

Negotiation flow

  1. 1
    Client connects
    ALPN, Content-Type, and Upgrade header inspected.
  2. 2
    Pick protocol filter
    REST · GraphQL · gRPC · WS handler is selected; matching credential extractor runs.
  3. 3
    Verify with IdP
    JWT signature, cert chain, or SPIFFE bundle.
  4. 4
    Normalize identity
    All claims projected to subject{ id, tenant, roles, scopes }.
  5. 5
    OPA policy decision
    Single Rego policy applied across protocols.
