Trigger

When the broker fires this webhook.

HTTP request

What the broker sends to your CI.

Headers

  • AuthorizationBearer ${GH_TOKEN}
  • Acceptapplication/vnd.github+json
  • X-GitHub-Api-Version2022-11-28

Body

{
  "ref": "main",
  "inputs": {
    "provider":        "${pactbroker.providerName}",
    "providerVersion": "${pactbroker.providerVersionNumber}",
    "consumer":        "${pactbroker.consumerName}",
    "consumerVersion": "${pactbroker.consumerVersionNumber}",
    "pactUrl":         "${pactbroker.pactUrl}"
  }
}

Tokens like ${pactbroker.providerVersionNumber} are substituted at fire time.
