obj --help

Cobra-style command tree
$ obj --help

obj — package manager for Objectified schemas (v0.7.4)

USAGE
  obj [global flags] <command> [command flags]

COMMANDS
  install, i        Install a package and its dependencies
  publish           Publish the current package to a registry
  search            Search the registry for packages
  login             Authenticate with the registry (browser OAuth)
  logout            Remove stored credentials
  whoami            Print the authenticated user
  ci                Strict-mode install for CI/CD pipelines
  cache             Manage the local package cache
  deprecate         Mark a version as deprecated
  licenses          Audit dependency licenses for compatibility
  audit             Check installed packages against the advisory feed

GLOBAL FLAGS
  --registry <url>  Override the registry URL
  --json            Emit machine-readable JSON
  --verbose         Enable debug logging
  --no-color        Disable ANSI colors

# Run obj <command> --help for command-specific options.
