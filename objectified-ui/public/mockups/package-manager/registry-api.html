Registry REST API

OpenAPI 3.1 · base https://registry.objectified.dev/api/v1

Endpoints

  • PUT /registry/{scope}/{name} publish version (multipart)
  • GET /registry/{scope}/{name} all versions metadata
  • GET /registry/{scope}/{name}/{version} single version + tarball URL
  • DELETE /registry/{scope}/{name}/{version} yank · 72h window
  • GET /registry/-/search?q=… full-text + facets · paginated
  • PUT /registry/{scope}/{name}/{version}/deprecate mark deprecated
  • POST /registry/tokens issue scoped token
  • GET /registry/-/advisories vulnerability feed
  • GET /registry/audit?from=… append-only event stream
