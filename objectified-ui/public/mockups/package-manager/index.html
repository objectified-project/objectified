Objectified Package Manager Mockup

npm/pip for schema packages · registry, CLI, discovery, and enterprise · Epics 1–4

Package Manager mockups

Visual designs for the Objectified Package Manager — a registry and CLI specifically for schema packages. Teams publish versioned .objpkg bundles, install them with deterministic dependency resolution, browse a public marketplace with quality and vulnerability signals, and run private registries with audit-grade compliance. Mockups follow the Studio design system: Inter typography, indigo accent, slate gray scale, panel-based layout, and dark mode parity.
