$ref resolution rules

live in the schema files inside schemas/
Localsame file
"$ref": "#/definitions/Money"

Resolved against the current schema document. No I/O required.

Intra-pkgsibling file
"$ref": "./customer.schema.json#/definitions/Address"

Relative path within the same package's schemas/ directory.

Cross-pkgnamespaced
"$ref": "@objectified/core-types/schemas/Money.schema.json"

Pkg name → resolved against installed dependency in obj_modules/.
