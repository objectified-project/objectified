    No obj-lock.json exists

    obj-package.json drift detected

  • Explicit update

    obj install --update

obj ci · strict-mode install for pipelines

Same as obj install but: refuses to mutate the lock, fails if manifest and lock disagree, never re-resolves.

  • Lock file must exist
  • Lock must match manifest hash exactly
  • Integrity hashes must match downloaded tarballs
  • Aborts on any unexpected resolution
