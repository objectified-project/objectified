Salesforce v4.2.1 Installed

By Objectified · 38 432 installs · 4.8 ★ (412 reviews) · Updated 4 days ago

Description

Bidirectional sync between Salesforce and Objectified schemas. Discover standard and custom objects, map to your domain model, and run incremental syncs every 5 minutes. Also supports Salesforce Streaming API (CometD) for real-time CDC.

Capabilities

Read

Standard + custom objects, SOQL filters, pagination.

Write

Insert / update / upsert with bulk API for > 1 K records.

Subscribe

CometD streams + Change Data Capture for < 5 s latency.

Version history

  • 4.2.14 days ago · Fix OAuth refresh under custom domain. Re-introduce Lead → Customer template.
  • 4.2.03 weeks ago · Add CometD subscribe for Account & Opportunity. Bump SF API to v60.
  • 4.1.02 months ago · Add Bulk API 2.0 for outbound writes > 10 K records.
  • 3.xMajor upgrade required to retain CometD subscriptions.

Authentication

OAuth 2.0 (Web Server)

Refresh token stored AES-256-GCM, auto-renewed at 80 % TTL.
