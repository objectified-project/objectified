Description

This change request introduces two improvements to the User entity:

  • Avatar URL: adds an optional avatar: string property for profile pictures (used by the new presence bar in collaboration).
  • Role enum: converts role: string into a proper enum with the four supported values: admin | editor | viewer | commenter. This unblocks the team-permissions epic.

Migration is non-breaking: existing string values map 1:1 to the enum. Generated TypeScript & Pydantic models update accordingly.
