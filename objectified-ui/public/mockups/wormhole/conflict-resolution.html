Selected for WH-001, WH-002

Source Wins

When a conflict is detected, the source environment's value always overrides the destination. Safest for one-way replication.

• Deterministic outcome
• No manual intervention
• Destination changes may be lost

Destination Wins

Source records are ignored if a newer version exists at the destination. Useful for protecting local overrides.

• Protects dest modifications
• Risk of source updates silently dropped

Manual Merge

Conflicted rows are placed in the merge queue. A human reviewer selects which version to retain or edits a merged value.

• Highest fidelity
• Requires operator involvement
• Backed by audit trail
