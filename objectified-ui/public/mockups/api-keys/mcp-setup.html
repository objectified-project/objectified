Client configuration snippets

Copy the config for your AI client

Add to ~/.cursor/mcp.json (or Cursor → Settings → MCP):

~/.cursor/mcp.json 
{
  "mcpServers": {
    "objectified": {
      "url": "https://mcp.objectified.io/mcp",
      "headers": {
        "Authorization": "Bearer odb_mcp_e1b8d9f2c4a7b3e6f1d8c9a0b5e2f3a4c7b8d9e0f1a2b3"
      }
    }
  }
}

Store the key in ~/.cursor/.env and reference it as ${OBJECTIFIED_MCP_KEY} to avoid committing secrets to source control.

Add to ~/Library/Application Support/Claude/claude_desktop_config.json:

claude_desktop_config.json 
{
  "mcpServers": {
    "objectified": {
      "command": "npx",
      "args": ["-y", "objectified-mcp"],
      "env": {
        "OBJECTIFIED_MCP_INTERNAL_SECRET": "odb_mcp_e1b8d9…",
        "OBJECTIFIED_MCP_URL": "https://mcp.objectified.io"
      }
    }
  }
}

After saving, restart Claude Desktop. The Objectified tools will appear in the tool selector.

Add to .vscode/settings.json in your workspace (or user settings):

.vscode/settings.json 
{
  "mcp.servers": {
    "objectified": {
      "type": "http",
      "url": "https://mcp.objectified.io/mcp",
      "headers": {
        "Authorization": "Bearer ${env:OBJECTIFIED_MCP_KEY}"
      }
    }
  }
}

Set OBJECTIFIED_MCP_KEY in your shell environment or in a .env file excluded from version control.

Use the fastmcp Python client or any MCP-compatible SDK:

agent.py 
import os
from fastmcp import Client

client = Client(
    "https://mcp.objectified.io/mcp",
    headers={
        "Authorization": f"Bearer {os.environ['OBJECTIFIED_MCP_KEY']}"
    },
)

async with client:
    specs = await client.call_tool("spec_list", {"tenant_id": "acme"})
    print(specs)

Install dependencies: pip install fastmcp
