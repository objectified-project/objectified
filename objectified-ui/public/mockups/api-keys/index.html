Design notes

  • Browser keys use indigo as the accent color; MCP Server keys use purple to provide instant visual distinction.
  • MCP key prefixes use the odb_mcp_ prefix per the existing odb.mcp_api_keys schema; Browser keys retain orcl_live_ / orcl_test_ prefixes.
  • The Create wizard shares a common base form and progressively reveals type-specific fields — no separate flows to maintain.
  • All dark/light theme toggles use localStorage for persistence, matching the existing mockup convention.
