Data Designer mockups

Visual designs for the Objectified schema designer — the React Flow canvas at /ade/studio/editor where teams draw their domain as a graph of classes wired together by $ref and link_def edges, edit JSON-Schema-style properties (composition, constraints, references), group related classes into lanes, freeze schema captures, and round-trip the same project as OpenAPI 3.1, JSON Schema, GraphQL SDL or SQL DDL through the Canvas | Code toggle. The mockups also expand on what the designer could become: a shared property library with usage analytics, first-class reference editing, the Studio AI co-pilot, and a versions panel that surfaces the immutable schema_capture timeline. Visual system reuses the existing Objectified mockup conventions: Inter typography, slate gray scale, panel cards, persisted dark mode.
